Kiqstnd is a prototype bike parking locator for use in City of Philadelphia! The entire version of this app was developed in a little over 12 hours -- which means that it's very rough and not ready for primetime. It was made to demostrate the possiblity of making a bike parking locator that uses real data provided by the City. Not only does it allow a user to search for bike parking locations, it also shows them thefts that have happened nearby a specific location. The other feature is to give users the ability to add bike parking locations themselves (think crowd sourcing).

Search Instructions (and viewing theft data):

  1. Start on the main [search] page
  2. Enter a Philadelphia-based address into the search box
  3. Click the Search button
  4. View the bike parking locations on the map
  5. Click on a map pin
  6. In the bubble that appears, click View Details
  7. The page that comes up shows theft locations on the map, as well as more details about the parking space

Add Parking Instructions:

  1. Start on the main [search] page
  2. Click on the map
  3. Fill out the form (not all of the fields will currently be saved)
  4. Click Ok
  5. Note: A pin will be placed, but as of right now, you'll have to refresh the page and search for the location again to access the View Details link on its bubble

Refresh the Bike Parking Data from the City

  1. Prepare yourself for the database to be wiped clean (don't worry, it'll be reloaded in the next step)
  2. Click here

The app is the result of collaboration between Brian Kenny, Brad Steinberg, and Jeff Maher (with guidance from Dave Walk) at the #apps4philly Transit Hackathon in conjunction with Code for Philly (a Code for America Brigade). Future development is planned and we're happy to work with others, so let us know if you want to join in. Or don't, and just fork the project on GitHub (warning: the code is very ugly right now).
